# 就活サポートアプリ

---

## 概要

- Webブラウザでカメラ撮影・診断・ホーム画面等から各種操作が可能
- HTML/CSS/JavaScript(Node.js/Express)によるシンプルな画面・API構成
- 教育・研究での利用を重視し、アクセシビリティや操作性も考慮
- 必要なファイル・設定は公開リポジトリ上で管理

---

## ファイル構成例

```
/home.html           ホーム画面
/camera.html         カメラ撮影画面
/sindan.html         診断（判定）画面
/css/style.css       スタイルシート
/js/main.js          メイン処理
/node_modules/       依存パッケージ
/server.js           APIサーバ（Node.js/Express）
/package.json        依存管理
/package-lock.json   依存管理
```
